# C++ Adventure Game

Written in C/C++ using the SDL Graphics Library,  
including lua scripting thanks to https://github.com/tomaka/luawrapper  

Features  
4 different playable areas  
Unique enemies for each zone  
2 dungeons  
4 choosable spells  
40+ items  
2 Quests  
Trading / Inventory system
